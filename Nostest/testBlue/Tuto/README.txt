Bon, explication de quoi fait quand vous voyez ce code. Déjà, installer Typescript. Tapez ça dans le cmd : npm install typescript --save-dev
Ensuite, vous allez sur VS code, vous faites clic droit sur tuto et "Ouvrir dans le terminal intégrer"
Vous tapez : npm install
Voila fini. Maintenant, pour lancer le script, vous tapez : npm run start et vous ouvrez une fenetre de navigateur avec l'url : http://localhost:8080/
